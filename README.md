# Nuxt

## Известные баги
- [ ] При выборе в таблице ряда, а потом ввода чего-то в поиск другие ряды выбираются
- [ ] По-хорошему бы в `app-popover-calendar` показывать значение `innerValue` пока календарь открыт
- [ ] Кнопка "Скачать" по таймауту отваливается
- [ ] При запросе >20 элементов на странице - может отвалиться запрос, на 50 100% отваливается

## Решения
Вся логика выносится в composables (`useSchools.ts`, `useRegions.ts`)
компоненты отвечают только за ui и какой-то стейт внутри себя
в страницах минимум логики

Остальная файловая структура по [официальной документации](https://nuxt.com/docs/guide)
